Cara Kerja Admin

Ketika admin setelah login, pilih Turn editing on untuk membuat survey pada moodle.

Kemudian setelah itu pilih Add an activity or resource. Untuk melanjutkan proses selanjutnya.

Langkah selanjutnya pilih Survey, kemudian pilih add, agar dapat melanjutkan langkah selanjutnya.

Setelah itu, langkah selanjutnya adalah isi nama materi untuk Survey, kemudian pilih Survey type, isi juga description dan yang terakhir checklist Display description on course page. 

Setelah semua terisi semua pilih save and display

Kemudian admin diharuskan memilih jawaban yang tersedia, yang nantinya digunakan untuk memberikan contoh cara memilih jawaban.

Setelah jawaban semua terisi, silahkan klik "Clik here to continue". Untuk melanjutkan proses selanjutnya.

Setelah itu klik "continue", untuk melanjutkan proses selanjutnya.

Tugas sebagai admin sudah selesai, kemudian log out akun.
